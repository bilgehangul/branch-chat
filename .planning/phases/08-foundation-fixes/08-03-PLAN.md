1. **Create backend/src/routes/config.ts**: A simple GET endpoint that returns the current model display name. This does NOT require auth (it's non-sensitive public config). Create a new file:
   ```typescript
   import { Router } from 'express';

   const PROVIDER = process.env.AI_PROVIDER ?? 'gemini';
   const MODEL_LABELS: Record<string, string> = {
     gemini: 'Gemini Flash 2.0',
     openai: 'GPT-4o',
   };

   export const configRouter = Router();

   configRouter.get('/', (_req, res) => {
     res.json({ modelLabel: MODEL_LABELS[PROVIDER] ?? PROVIDER });
   });
   ```

2. **backend/src/routes/index.ts**: Mount the config router BEFORE the auth middleware so it's publicly accessible (model label is not sensitive). Add ABOVE `apiRouter.use(requireApiAuth)`:
   ```typescript
   import { configRouter } from './config.js';
   // Public routes (no auth required)
   apiRouter.use('/config', configRouter);
   // Auth guard on remaining routes
   apiRouter.use(requireApiAuth);
   ```
   Wait — actually all routes in apiRouter go through requireApiAuth. The config endpoint should be accessible without auth. Options: (a) mount it on the parent Express app directly, or (b) mount it before the requireApiAuth middleware. Option (b) is simpler — Express middleware runs in order, so routes mounted before `apiRouter.use(requireApiAuth)` won't require auth. Place the config mount BEFORE the auth middleware line.

3. **Frontend — fetch model label**: In MessageBlock.tsx, fetch the model label once at the module level or use a simple React state + useEffect approach. Since MessageBlock is rendered many times per thread, avoid fetching in each instance. Better approach: create a tiny module-level cache.

   Create a small utility `frontend/src/api/config.ts`:
   ```typescript
   let cachedLabel: string | null = null;

   export async function getModelLabel(): Promise<string> {
     if (cachedLabel) return cachedLabel;
     try {
       const res = await fetch('/api/config');
       const data = await res.json();
       cachedLabel = data.modelLabel ?? 'AI';
       return cachedLabel;
     } catch {
       return 'AI';
     }
   }
   ```

4. **MessageBlock.tsx**: Replace the hardcoded `'Gemini'` (line 45) with a dynamic label. Use a `useState` + `useEffect` at the component level. Since ALL instances share the same label, the module-level cache means only the first render triggers a fetch:
   ```tsx
   const [modelLabel, setModelLabel] = useState('AI');
   useEffect(() => {
     getModelLabel().then(setModelLabel);
   }, []);
   ```
   Then on line 45: `{isUser ? 'You' : modelLabel}`

   Import `getModelLabel` from `../../api/config`.
