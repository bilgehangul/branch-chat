<!-- Pill positioning approach (from 04-RESEARCH.md Pattern 3) -->
```typescript
function measurePillTop(
  messageId: string,
  paragraphId: string,
  scrollContainer: HTMLElement
): number | null {
  const anchor = scrollContainer.querySelector(
    `[data-message-id="${messageId}"] [data-paragraph-id="${paragraphId}"]`
  );
  if (!anchor) return null;
  const anchorRect = anchor.getBoundingClientRect();
  const containerRect = scrollContainer.getBoundingClientRect();
  return anchorRect.top - containerRect.top + scrollContainer.scrollTop;
}
```

Implementation structure:

1. Early return: `if (activeThread.childThreadIds.length === 0) return null;`

2. Collect childLeads from all messages in the active thread:
```typescript
const childLeads = activeThread.messageIds
  .flatMap(id => messages[id]?.childLeads ?? []);
```

3. Pill position tracking:
```typescript
const pillPositions = useRef<Record<string, number>>({});
const [posVersion, setPosVersion] = useState(0); // triggers re-render when positions change

function recomputePositions() {
  const container = scrollContainerRef.current;
  if (!container) return;
  let changed = false;
  for (const lead of childLeads) {
    const top = measurePillTop(/* find messageId containing this lead */, String(lead.paragraphIndex), container);
    if (top !== null) {
      const prev = pillPositions.current[lead.threadId];
      if (prev === undefined || Math.abs(prev - top) > 1) {
        pillPositions.current[lead.threadId] = top;
        changed = true;
      }
    }
  }
  if (changed) setPosVersion(v => v + 1);
}
```

For `measurePillTop`, the gutter needs to know which messageId contains each childLead. Find it:

```typescript
// Find the messageId that has this childLead
```

4. ResizeObserver setup:
```typescript
useEffect(() => {
  recomputePositions(); // initial measurement
  if (typeof ResizeObserver === 'undefined') return;
  const ro = new ResizeObserver(() => {
    requestAnimationFrame(recomputePositions);
  });
  if (scrollContainerRef.current) ro.observe(scrollContainerRef.current);
  return () => ro.disconnect();
}, [childLeads.length, posVersion]); // re-run when leads added or scroll happens
```

Also add scroll listener on container to recompute (pills are position:absolute relative to container):
```typescript
useEffect(() => {
  const el = scrollContainerRef.current;
  if (!el) return;
  const handleScroll = () => requestAnimationFrame(recomputePositions);
  el.addEventListener('scroll', handleScroll, { passive: true });
  return () => el.removeEventListener('scroll', handleScroll);
}, []);
```
